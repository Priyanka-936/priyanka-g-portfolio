# Priyanka G Portfolio

A modern personal portfolio built with React, Vite, Framer Motion, and Tailwind CSS.

## Features

* Professional recruiter-focused design
* Animated Hero Section
* About Me
* Skills
* Projects
* Internships
* Certifications
* Education Timeline
* Contact Section
* PRIYA AI Assistant

## Tech Stack

* React
* Vite
* Tailwind CSS
* Framer Motion
